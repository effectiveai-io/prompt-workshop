# TC 설계 평가 기준

각 항목 0~2점. 0: 미반영, 1: 부분 반영, 2: 완전 반영. 만점 10점.

## 평가 항목

### 1. 시나리오 커버리지 (0~2점)
- 0점: 정상 시나리오만 있음
- 1점: 정상 + 일부 비정상 포함
- 2점: 정상, 비정상, 경계값이 모두 포함됨

### 2. 우선순위 분류 (0~2점)
- 0점: 우선순위 없이 나열
- 1점: 일부 TC에만 우선순위 표시
- 2점: 모든 TC에 P1/P2/P3 우선순위가 명확히 분류됨

### 3. 재현 가능성 (0~2점)
- 0점: "로그인하고 테스트" 수준의 모호한 단계
- 1점: 단계는 있지만 사전 조건이 불명확
- 2점: 사전 조건, 단계, 기대 결과가 모두 구체적이어서 누구나 재현 가능

### 4. 비즈니스 맥락 반영 (0~2점)
- 0점: 기능 명세만 따라감
- 1점: 일부 도메인 규칙이 반영됨
- 2점: 금융/규제/보안 등 도메인 특화 시나리오가 포함됨

### 5. 형식 일관성 (0~2점)
- 0점: TC마다 형식이 제각각
- 1점: 대부분 일관되지만 일부 누락
- 2점: 모든 TC가 TC ID, 우선순위, 사전조건, 단계, 기대결과를 포함
