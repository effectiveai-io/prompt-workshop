# 문서 초안 평가 기준

각 항목 0~2점. 0: 미반영, 1: 부분 반영, 2: 완전 반영. 만점 10점.

## 평가 항목

### 1. 구조 완결성 (0~2점)
- 0점: 제목과 본문만 있음
- 1점: 일부 섹션 구조가 있지만 불완전
- 2점: 요약, 배경, 본문, 결론이 모두 포함되고 목차로 파악 가능

### 2. 독자 수준 적합성 (0~2점)
- 0점: 대상 독자를 고려하지 않은 용어 사용
- 1점: 일부 전문 용어를 설명
- 2점: 대상 독자의 수준에 맞고, 전문 용어는 모두 첫 등장 시 설명

### 3. 근거와 예시 (0~2점)
- 0점: 주장만 있고 근거 없음
- 1점: 일부 근거나 예시가 포함
- 2점: 모든 핵심 주장에 구체적 수치, 사례, 코드 예시가 뒷받침됨

### 4. 마크다운 형식 (0~2점)
- 0점: 서식 없는 텍스트 나열
- 1점: 헤딩은 있지만 코드 블록, 표 등 활용 부족
- 2점: 헤딩, 코드 블록(언어 표시), 표, 인용 등 마크다운을 효과적으로 활용

### 5. 실행 가능한 결론 (0~2점)
- 0점: 결론 없이 끝남
- 1점: 요약은 있지만 다음 단계가 불명확
- 2점: 누가 무엇을 언제까지 해야 하는지 명확한 액션 아이템 포함
