# 문서 초안 평가 기준

10개 항목, 각 1~5점. 만점 50점.

**채점 원칙**: 각 점수의 기준은 "결과물에서 세거나 확인할 수 있는 것"입니다. AI의 주관적 판단이 아니라, 결과물에 해당 내용이 있는지 없는지로 채점합니다.

## 평가 항목

### 1. 구조 완결성 (1~5)
- 1점: 제목과 본문만 있음 (섹션 구분 없음)
- 2점: 헤딩이 2개 이상 있지만 요약/결론 중 하나 이상 누락
- 3점: 요약, 배경, 본문, 결론 4개 섹션이 모두 존재
- 4점: 4개 섹션 + 목차만 보고 전체 흐름 파악 가능
- 5점: 4개 섹션 + 목차 + 각 섹션의 분량이 내용에 맞게 균형

### 2. 한 줄 요약 (1~5)
- 1점: 요약이 없음
- 2점: 요약이 있지만 2줄 이상이거나 핵심이 불명확
- 3점: 1줄 요약이 있고 문서의 핵심 메시지가 담겨있음
- 4점: 1줄 요약 + 요약만 읽고도 "이 문서를 읽어야 하는지" 판단 가능
- 5점: 요약 + 대상 독자 + 예상 소요 시간이 명시됨

### 3. 배경/맥락 (1~5)
- 1점: 배경 설명 없이 바로 본문 시작
- 2점: "이 문서의 목적은~" 1줄 수준
- 3점: 왜 필요한지 + 현재 상태가 설명됨 (2문단 이상)
- 4점: 필요성 + 현재 상태 + 이 문서가 해결하려는 문제가 명시됨
- 5점: 문제 + 이해관계자 + 의사결정에 필요한 맥락까지 포함

### 4. 독자 수준 적합성 (1~5)
- 1점: 설명 없는 전문 용어가 3개 이상
- 2점: 전문 용어 설명이 일부만 있음 (50% 미만)
- 3점: 전문 용어 첫 등장 시 설명이 80% 이상 포함
- 4점: 용어 설명 + 약어 풀어쓰기까지 포함
- 5점: 용어 + 약어 + 독자별 읽기 가이드 (예: "개발자는 3장부터")

### 5. 근거와 수치 (1~5)
- 1점: 주장에 근거/수치가 0건
- 2점: "~한 것으로 알려져 있다" 같은 출처 없는 언급만 있음
- 3점: 구체적 수치 또는 사례가 1건 이상 포함
- 4점: 핵심 주장의 50% 이상에 수치/사례/출처가 뒷받침됨
- 5점: 수치 + 출처 + 수치의 한계/가정 조건까지 명시

### 6. 코드/기술 예시 (1~5)
- 1점: 기술 문서인데 코드 블록이 0개
- 2점: 코드 블록이 있지만 언어 표시가 없거나 의사코드 수준
- 3점: 언어 표시된 실행 가능한 코드가 1개 이상
- 4점: 코드 + 설명 주석 + 실행 결과 예시 포함
- 5점: 코드 + 결과 + 에러 케이스 + "이렇게 하면 안 되는" 안티패턴

### 7. 시각적 설명 (1~5)
- 1점: 텍스트만으로 나열 (표/다이어그램 0개)
- 2점: 표가 1개 이상 포함
- 3점: 표 + 다이어그램/플로우차트가 1개 이상 포함 (또는 필요 표시)
- 4점: 시각 자료 + 각 자료에 캡션/설명이 포함됨
- 5점: 시각 자료 + 캡션 + "이 다이어그램이 설명하는 핵심" 1줄 명시

### 8. 실행 가능한 결론 (1~5)
- 1점: 결론 없이 끝남
- 2점: 요약은 있지만 "다음에 할 일"이 없음
- 3점: "다음에 이런 걸 해야 한다" 방향이 1건 이상
- 4점: 액션 아이템이 2건 이상 + 각각에 담당자 또는 기한 포함
- 5점: 액션 아이템 + 담당자 + 기한 + 성공 기준까지 명시

### 9. 마크다운 활용 (1~5)
- 1점: 서식 없는 텍스트 나열
- 2점: 헤딩(#)만 사용
- 3점: 헤딩 + 리스트 + 코드 블록 사용
- 4점: 헤딩 + 리스트 + 코드 블록(언어 표시) + 표 + 인용(>) 사용
- 5점: 전체 마크다운 요소 + Callout/강조 + 접기/펼치기 활용

### 10. 유지보수 가능성 (1~5)
- 1점: 날짜, 작성자, 버전 표시가 없음
- 2점: 날짜 또는 작성자 중 하나만 있음
- 3점: 날짜 + 작성자 + 버전 표시가 있음
- 4점: 날짜/작성자/버전 + 관련 문서 링크 + 용어 정의 포함
- 5점: 위 전부 + 변경 이력 + "이 문서가 outdated되는 조건" 명시

## 예상 점수대

| 프롬프트 수준 | 예상 점수 | 설명 |
|---|---|---|
| 빈 프롬프트 | 12~18점 | 기본 문서 |
| 기본 구조 포함 | 20~26점 | 섹션 구조, 마크다운 |
| 독자/맥락 포함 | 27~34점 | 배경, 근거, 용어 |
| 전문가 수준 | 34~42점 | 시각화, 액션, 유지보수 |
| 만점 50점 | 사실상 불가능 | 모든 항목 5점은 매우 어려움 |
