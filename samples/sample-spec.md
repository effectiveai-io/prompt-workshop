# 샘플 기획서 — 자동 매수 예약 기능

아래 기획서를 리뷰 대상으로 사용합니다.

---

## 기능 개요

사용자가 특정 가격에 도달하면 자동으로 매수가 실행되는 예약 기능을 추가합니다.

## 배경

사용자들이 가격을 계속 모니터링하면서 수동으로 매수하는 불편함이 있습니다.
경쟁사에서는 이미 제공하고 있는 기능이며, CS 문의 중 10%가 관련 요청입니다.

## 상세 기능

### 예약 등록
- 사용자가 종목, 목표 가격, 매수 수량을 입력합니다
- 예약이 등록되면 대기 상태가 됩니다
- 적절한 시점에 알림을 발송합니다

### 자동 체결
- 시장 가격이 목표 가격에 도달하면 자동으로 매수 주문을 실행합니다
- 체결 후 사용자에게 알림을 발송합니다

### 예약 관리
- 사용자가 예약 목록을 조회할 수 있습니다
- 예약을 취소할 수 있습니다
