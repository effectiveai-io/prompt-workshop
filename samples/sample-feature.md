# 샘플 기능 설명 — 출금 주소 화이트리스트

아래 기능 설명을 TC 설계 또는 문서 초안 작성의 입력으로 사용합니다.

---

## 기능 설명

사용자가 미리 등록한 주소로만 출금이 가능하도록 하는 화이트리스트 기능입니다.

### 주소 등록
- 사용자가 출금 주소를 등록할 수 있습니다
- 등록 시 주소 라벨(별칭)을 함께 입력합니다
- 등록 후 24시간 대기 기간이 있습니다 (보안)
- 대기 기간 중에는 해당 주소로 출금 불가

### 주소 삭제
- 등록된 주소를 삭제할 수 있습니다
- 삭제 시 본인 인증(2FA)이 필요합니다

### 출금 시 검증
- 출금 요청 시 대상 주소가 화이트리스트에 있는지 확인합니다
- 화이트리스트에 없는 주소로의 출금은 차단됩니다
- 대기 기간 중인 주소로의 출금도 차단됩니다

### 화이트리스트 ON/OFF
- 사용자가 화이트리스트 기능 자체를 켜고 끌 수 있습니다
- OFF 상태에서는 모든 주소로 출금 가능 (기존 방식)
- ON→OFF 전환 시 24시간 대기 기간 적용

## 참고 맥락

- 암호화폐 거래소의 보안 기능
- 해킹 피해 방지가 주된 목적
- 최대 등록 가능 주소: 20개
- 네트워크별로 주소 형식이 다름 (BTC, ETH, TRX 등)
